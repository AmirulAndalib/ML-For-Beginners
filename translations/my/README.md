# စတင်သူများအတွက် စက်လေ့လာမှု - သင်ရိုးညွှန်းတမ်း

> 🌍 ကမ္ဘာ့ယဉ်ကျေးမှုများမှတဆင့် စက်လေ့လာမှုကို လေ့လာရင်း ကမ္ဘာတစ်ဝှမ်း ခရီးသွားကြစို့ 🌍

Microsoft ၏ Cloud Advocates များသည် **စက်လေ့လာမှု** အကြောင်းကို 12 ပတ်ကြာ 26 သင်ခန်းစာပါ သင်ရိုးညွှန်းတမ်းတစ်ခုကို ပေးဆောင်ရန် ဝမ်းမြောက်စွာ တင်ဆက်လိုက်ပါသည်။ ဤသင်ရိုးညွှန်းတမ်းတွင် **စက်လေ့လာမှု၏ အခြေခံပုံစံများ** ကို Scikit-learn ကို အဓိကအသုံးပြု၍ သင်ယူမည်ဖြစ်ပြီး၊ [AI for Beginners' curriculum](https://aka.ms/ai4beginners) တွင် ဖော်ပြထားသည့် နက်ရှိုင်းသောလေ့လာမှုများကို မပါဝင်စေပါ။

ကမ္ဘာတစ်ဝှမ်းရှိ ဒေတာများကို အသုံးပြု၍ ဤအခြေခံနည်းလမ်းများကို လေ့ကျင့်ရင်း ကျွန်ုပ်တို့နှင့်အတူ ခရီးသွားပါ။ သင်ခန်းစာတိုင်းတွင် သင်ခန်းစာမတိုင်မီနှင့် သင်ခန်းစာပြီးနောက် စစ်ဆေးမှုများ၊ သင်ခန်းစာကို ပြီးစီးရန် ရေးသားထားသော လမ်းညွှန်ချက်များ၊ ဖြေရှင်းချက်၊ လုပ်ငန်းတာဝန်များနှင့် အခြားအရာများ ပါဝင်သည်။

# စတင်ရန်

ဤအဆင့်များကို လိုက်နာပါ:
1. **Repository ကို Fork လုပ်ပါ**: ဤစာမျက်နှာ၏ အပေါ်ညာဘက်ထောင့်ရှိ "Fork" ခလုတ်ကို နှိပ်ပါ။
2. **Repository ကို Clone လုပ်ပါ**: `git clone https://github.com/microsoft/ML-For-Beginners.git`

> [ဤသင်ရိုးညွှန်းတမ်းအတွက် အပိုဆောင်းအရင်းအမြစ်များကို Microsoft Learn collection တွင် ရှာဖွေပါ](https://learn.microsoft.com/en-us/collections/qrqzamz1nn2wx3?WT.mc_id=academic-77952-bethanycheum)

**[ကျောင်းသားများ](https://aka.ms/student-page)**, ဤသင်ရိုးညွှန်းတမ်းကို သင်၏ GitHub အကောင့်သို့ Fork လုပ်ပြီး သင်တစ်ဦးတည်းဖြစ်စေ၊ အဖွဲ့ဖြင့်ဖြစ်စေ လေ့ကျင့်ပါ:

- သင်ခန်းစာမတိုင်မီ စစ်ဆေးမှုဖြင့် စတင်ပါ။
- သင်ခန်းစာကို ဖတ်ပြီး လုပ်ငန်းဆောင်တာများကို ပြီးစီးပါ။
- ဖြေရှင်းချက်ကို မရိုက်မီ သင်ခန်းစာများကို နားလည်ရန် ကြိုးစားပါ။
- သင်ခန်းစာပြီးနောက် စစ်ဆေးမှုကို ပြုလုပ်ပါ။
- စိန်ခေါ်မှုကို ပြီးစီးပါ။
- လုပ်ငန်းတာဝန်ကို ပြီးစီးပါ။
- သင်ခန်းစာအုပ်စုတစ်ခုကို ပြီးစီးပြီးနောက် [Discussion Board](https://github.com/microsoft/ML-For-Beginners/discussions) သို့ သွားပါ။

> ပိုမိုလေ့လာရန် [Microsoft Learn](https://docs.microsoft.com/en-us/users/jenlooper-2911/collections/k7o7tg1gp306q4?WT.mc_id=academic-77952-leestott) ကို လိုက်နာရန် အကြံပြုပါသည်။

**ဆရာများ**, ဤသင်ရိုးညွှန်းတမ်းကို အသုံးပြုရန် အကြံပြုချက်များကို [ဤနေရာတွင်](for-teachers.md) ထည့်သွင်းထားပါသည်။

## သင်ကြားမှုနည်းလမ်း

ဤသင်ရိုးညွှန်းတမ်းကို ဖန်တီးစဉ်တွင် **ပရောဂျက်အခြေပြု** နှင့် **မကြာခဏ စစ်ဆေးမှုများ** ပါဝင်စေရန် ရည်ရွယ်ထားပါသည်။ ထို့အပြင် သင်ရိုးညွှန်းတမ်းတွင် **အဓိကအကြောင်းအရာ** တစ်ခုကို ထည့်သွင်းထားသည်။

ပရောဂျက်များနှင့် အညီ အကြောင်းအရာများကို သင်ကြားခြင်းဖြင့် ကျောင်းသားများအတွက် ပိုမိုစိတ်ဝင်စားစေပြီး သင်ခန်းစာများကို ပိုမိုမှတ်မိစေပါသည်။ 

> ကျွန်ုပ်တို့၏ [Code of Conduct](CODE_OF_CONDUCT.md), [Contributing](CONTRIBUTING.md), နှင့် [Translation](TRANSLATIONS.md) လမ်းညွှန်ချက်များကို ရှာဖွေပါ။
