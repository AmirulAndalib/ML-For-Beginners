# ਸਹਾਇਤਾ
## ਮਸਲੇ ਦਰਜ ਕਰਨ ਅਤੇ ਮਦਦ ਪ੍ਰਾਪਤ ਕਰਨ ਦਾ ਤਰੀਕਾ  

ਮਸਲਾ ਦਰਜ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ, ਕਿਰਪਾ ਕਰਕੇ ਸਥਾਪਨਾ, ਸੈਟਅਪ ਅਤੇ ਪਾਠਾਂ ਚਲਾਉਣ ਨਾਲ ਸੰਬੰਧਿਤ ਆਮ ਸਮੱਸਿਆਵਾਂ ਲਈ ਸਾਡੇ [Troubleshooting Guide](TROUBLESHOOTING.md) ਦੀ ਜਾਂਚ ਕਰੋ।

ਇਹ ਪ੍ਰੋਜੈਕਟ ਬੱਗਸ ਅਤੇ ਫੀਚਰ ਦੀਆਂ ਬੇਨਤੀਆਂ ਨੂੰ ਟ੍ਰੈਕ ਕਰਨ ਲਈ GitHub Issues ਦੀ ਵਰਤੋਂ ਕਰਦਾ ਹੈ। ਨਵੇਂ ਮਸਲੇ ਦਰਜ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਮੌਜੂਦਾ ਮਸਲਿਆਂ ਦੀ ਖੋਜ ਕਰੋ ਤਾਂ ਜੋ ਦੁਹਰਾਅ ਤੋਂ ਬਚਿਆ ਜਾ ਸਕੇ। ਨਵੇਂ ਮਸਲਿਆਂ ਲਈ, ਆਪਣਾ ਬੱਗ ਜਾਂ ਫੀਚਰ ਦੀ ਬੇਨਤੀ ਇੱਕ ਨਵੇਂ Issue ਵਜੋਂ ਦਰਜ ਕਰੋ।

ਇਸ ਪ੍ਰੋਜੈਕਟ ਦੀ ਵਰਤੋਂ ਬਾਰੇ ਮਦਦ ਅਤੇ ਸਵਾਲਾਂ ਲਈ, ਤੁਸੀਂ ਇਹ ਵੀ ਕਰ ਸਕਦੇ ਹੋ:
- [Troubleshooting Guide](TROUBLESHOOTING.md) ਦੀ ਜਾਂਚ ਕਰੋ
- ਸਾਡੇ [Discord Discussions #ml-for-beginners channel](https://aka.ms/foundry/discord) 'ਤੇ ਜਾਓ
- ਇੱਕ ਮਸਲਾ ਦਰਜ ਕਰੋ
